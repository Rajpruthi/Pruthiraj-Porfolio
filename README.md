# Pruthiraj Portfolio

A modern React-based portfolio website showcasing projects, skills, and professional information.

## Features

- Responsive design using Tailwind CSS
- Multiple project pages including:
  - Car Price Prediction
  - Movie Recommendation Engine
  - Bulk Email Sender
  - Tourism Management
  - NDT & DT Testing
- Interactive UI with smooth animations
- Contact form functionality

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

1. Clone the repository:
```
bash
git clone https://github.com/Rajpruthi/Pruthiraj-Porfolio.git
```

2. Install dependencies:
```
bash
npm install
```

3. Run the development server:
```
bash
npm run dev
```

4. Build for production:
```
bash
npm run build
```
